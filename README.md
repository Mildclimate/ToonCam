# ToonCam

ToonCam is a real-time cartoon camera app built with React Native + Expo.

## Structure

- `app/` Expo Router screens and layout
- `src/components/` shared UI components
- `src/features/` feature-based modules
- `src/hooks/` reusable hooks
- `src/services/` platform services
- `src/store/` global state
- `src/styles/` theme and style tokens
- `assets/` images, icons, fonts
- `.vscode/` workspace settings and Copilot support
- `skills/` reusable task knowledge
- `agents/` agent-specific notes and prompts

## Scripts

- `npm run start`
- `npm run ios`
- `npm run android`
- `npm run web`
- `npm run lint`
- `npm run typecheck`
