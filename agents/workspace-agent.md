# Workspace Agent Prompt

Use this prompt when asking an agent to extend ToonCam.

## Goal

Make the smallest useful code change that moves the project toward a working MVP.

## Preferred Workflow

1. Read `plan.md` and `copilot.md`.
2. Find the closest existing implementation.
3. Make a minimal change.
4. Validate the touched slice.
5. Stop after the requirement is satisfied.
