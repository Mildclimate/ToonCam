# Realtime Camera Filter Skill

## Purpose

Guide the LLM to implement ToonCam camera preview, frame processing, and filter UI with minimal and testable changes.

## Rules

- Prefer the existing Expo + React Native + TypeScript stack.
- Keep processing code inside `src/features/camera/`.
- Start with a simple working path before optimizing performance.
- Use 720p-first thinking for MVP performance.

## Typical Output

- camera permission hook
- preview component
- filter mode state
- frame processor stub or implementation
